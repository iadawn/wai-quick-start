Quick Start Tips

These guides provide you with practical tips to get started with web accessibility, and pointers to detailed resources to learn more.

  • Designing: Quick start tips for accessible visual and interface design.
  • Developing: Quick start tips for accessible coding.
  • Authoring: Quick start tips for writing accessible content for the web.
  • Evaluating: Quick start tips for website accessibility evaluation process.
  • Managing: Quick start tips for planning and managing web accessibility.
  • Advocating: Quick start tips for championing accessibility.
